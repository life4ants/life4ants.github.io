About Me

Hello, I'm Andrew Buck. I'm a homeschooled graduate who continues to learn everyday. My interests include languages, history, physics, and economics.

I've dabbled in web development a lot as a hobby but I'm not sure if I want to seek it as a carrer.

Besides the usual HTML/CSS/JavaScript(ES6), I've worked with JQuery, Vue, React, p5.js, and Ruby on Rails.

In the last few years I've played with hardware on the most basic level, Ben Eater style. The second version of my 8-bit computer was never finished because it became too complicated to trouble shoot, but it was fun while it lasted.

My biggest project has been my Wemo Explorer game build on p5.js and Vue.js which can be found on Andysgames.fun
